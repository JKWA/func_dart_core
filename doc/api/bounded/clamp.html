Returns a function that clamps a value of type A within its bounds.

The function will return the bottom if the value is less than the lower bound and the top if it is greater than the upper bound. If the value is within the bounds, it will be returned as is.

Implementation

A Function(A) clamp<A>(Bounded<A> bounded) {
  return (A a) {
    if (bounded.compare(a, bounded.bottom) < 0) {
      return bounded.bottom;
    } else if (bounded.compare(a, bounded.top) > 0) {
      return bounded.top;
    } else {
      return a;
    }
  };
}
